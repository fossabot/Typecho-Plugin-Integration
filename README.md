# Typecho-Plugin-Integration

基于Typecho博客插件整合的一个小插件！

## 安装

第 1 步：下载本插件，解压，放到 usr/plugins/ 目录中；

第 2 步：文件夹名改为 `Integration`；

第 3 步：登录管理后台，激活插件

## 升级方法

**请先禁用插件后再升级**
